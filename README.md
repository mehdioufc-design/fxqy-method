# FXQY Method

FXQY Method is a Windows desktop video optimizer that processes videos locally using the user's own CPU or supported GPU. Videos are not uploaded to a processing server.

## Download

Download the latest Windows installer from the Releases page: https://github.com/mehdioufc-design/fxqy-method/releases/latest

## Windows notice

The current installer is not digitally signed. Microsoft Smart App Control may block unsigned downloads. A trusted Authenticode code-signing certificate is required to remove the unknown-publisher warning for public distribution.

## Privacy

Processing, accounts, settings, uploads, and exports remain on the installed computer.
